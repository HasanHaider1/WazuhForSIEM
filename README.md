# WazuhForSIEM
## Item(s)
Security report 
## Tools
1. Wazuh
2. Ubuntu
3. Windows.
## Objectives
1. Monitor a remote system for file integrity
2. Get real time updates for attempted cyber-attacks (Brute-force, in this case)
## Limitations
1. Works on known threats and lacks Zero Day Detection Capabilities
2. Works in binary detection in isolation without behavior recognition or contextual awareness
3. Requires manual viewing of logs
